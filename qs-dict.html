About:

This is not a true dictionary, but rather a phonetic transformation from english words to Quickscript. Some standardized spellings may be transformed incorrectly. Submit any suggestions as an issue on this project's github page.

The "Phenomes" section displayes the phenomes obtained from the Carnegie Mellon Pronouncing Dictionary. For more information on these phenomes, see the ARPABET wikipedia article.
